# Forward Bot (Telethon + Aiogram)

Простой управляемый бот для пересылки сообщений из Telegram-каналов и групп в целевой бот или чат.

## Что осталось
- Источники: только каналы, группы и супергруппы аккаунта Telethon.
- Пересылка: видео пересылаются напрямую через `forward_messages`, без скачивания файлов.
- Фильтр видео: пересылаются только вертикальные видео, которые Telegram отдаёт как `msg.video` или документ с MIME-типом `video/*`, и которые проходят ограничения по разрешению, размеру и длительности; остальные сообщения пропускаются.
- Защита от дублей: уже пересланные сообщения сохраняются в SQLite и повторно не отправляются.
- Управление через Bot API: нижнее меню Telegram, выбор источников, удаление источника из проверок, статистика видео по чатам, запуск по всем источникам, период, порядок, статус и мягкая остановка.
- Логи в консоль и файл `logs/app.log`.
- Heartbeat и watchdog для контроля зависаний.

## Режимы пересылки
- `new` — только сообщения новее последней сохранённой позиции по каждому источнику.
- `day` — сообщения за последние 24 часа.
- `week` — сообщения за последние 7 дней.
- `month` — сообщения за последние 30 дней.
- `all` — вся доступная история выбранных каналов и групп.

## Меню управления
- `🔎 Все` — переслать из всех доступных каналов и групп.
- `📌 Выбрать` — отметить конкретные источники и запустить только их; кнопка `🗑` исключает чат из дальнейших проверок.
- `⏱ Период` — выбрать глубину истории.
- `🔁 Порядок` — новые → старые или старые → новые.
- `📄 Статус` — текущий процесс, heartbeat и последний итог.
- `📊 Чаты` — список всех доступных чатов с количеством видео за выбранный период и количеством подходящих видео, которые ещё не пересылались.
- `⛔ Стоп` — мягко остановить текущую пересылку.

Доступны команды `/start`, `/status` и `/help`.

## Установка
```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
copy .env.example .env
python main.py
```

## Конфигурация
Скопируй `.env.example` в `.env` и заполни значения:

- `BOT_TOKEN` — токен управляющего Bot API бота.
- `API_ID` / `API_HASH` — MTProto параметры аккаунта Telegram.
- `PHONE` — телефон аккаунта для первичной авторизации Telethon.
- `TG_2FA_PASSWORD` — пароль 2FA, если включён.
- `TARGET_BOT_USERNAME` — username целевого бота или чата для пересылки, без `@` или с ним.
- `ALLOWED_USERS` — необязательный CSV-список Telegram user id, которым разрешено управлять ботом.
- `EXCLUDED_CHAT_IDS` — необязательный CSV-список id каналов/групп, которые нужно пропустить. Также автоматически пропускаются чаты, в названии которых есть `vertical`, и источники, удалённые через кнопку `🗑`.
- `VIDEO_VERTICAL_ONLY` — пересылать только вертикальные видео (`true` по умолчанию).
- `VIDEO_EXACT_WIDTH` / `VIDEO_EXACT_HEIGHT` — точное разрешение видео; по умолчанию `0` × `0`, то есть точная проверка отключена, пока ты сам не задашь нужные значения.
- `VIDEO_MIN_WIDTH` / `VIDEO_MIN_HEIGHT` / `VIDEO_MAX_WIDTH` / `VIDEO_MAX_HEIGHT` — дополнительные границы разрешения; по умолчанию ширина от `900px`, `0` отключает соответствующую границу.
- `VIDEO_MIN_DURATION_SEC` / `VIDEO_MAX_DURATION_SEC` — границы длительности в секундах; по умолчанию длительность от `180` секунд (3 минуты), максимум отключён (`0`).
- `VIDEO_MIN_SIZE_MB` / `VIDEO_MAX_SIZE_MB` — границы общего размера файла в мегабайтах; по умолчанию отключены (`0`).
- `VIDEO_MIN_SIZE_MB_PER_MINUTE` — минимальный размер на минуту видео; по умолчанию `10` МБ за 1 минуту.

Локальный `.env`, сессии, базы и логи должны оставаться вне git.

## Логи
- `logs/app.log`
- смотреть “хвост” в PowerShell:
```powershell
Get-Content .\logs\app.log -Tail 200 -Wait
```

## Важно
Перед первым запуском открой целевой бот/чат с аккаунта Telethon и убедись, что аккаунту разрешено туда отправлять сообщения.
